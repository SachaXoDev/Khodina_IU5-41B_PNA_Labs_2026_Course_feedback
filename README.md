# Лабораторная работа №3
## Простое веб-приложение: Обратная связь по курсу

**Студент:** Ходина Александра Павловна  
**Группа:** ИУ5-41Б  

## Задание

1. Создать простое веб-приложение для отображения информации о группах помощи студентам
2. Реализовать карточки групп с информацией
3. Добавить страницу детального просмотра
4. Реализовать функционал добавления, удаления и фильтрации групп
5. Добавить 3D модель для визуализации

## Технологии

- **HTML5** - структура страниц
- **CSS3** - стилизация и анимации
- **JavaScript (ES6+)** - логика приложения
- **Bootstrap 5** - адаптивный дизайн
