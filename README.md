# Лабораторная работа №4
## Создание REST API на Express.js для управления группами

**Студент:** Ходина Александра Павловна  
**Группа:** ИУ5-41Б  

## Задание

1. Создать REST API для управления группами помощи студентам
2. Реализовать полный CRUD функционал
3. Хранение данных в JSON файле
4. Подготовить API для интеграции с фронтендом

## Технологии

- **Node.js** - среда выполнения
- **Express.js** - веб-фреймворк
- **CORS** - кросс-доменные запросы
- **File System (fs)** - работа с JSON файлом
- **nodemon** - автоматический перезапуск сервера
