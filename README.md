# Лабораторная работа №1
## Создание макета калькулятора

**Студент:** Ходина Александра Павловна  
**Группа:** ИУ5-41Б  

## Задание

1. Создать макет калькулятора с помощью HTML и CSS
2. Реализовать сетку кнопок
3. Добавить дисплей для отображения ввода и результата
4. Обеспечить адаптивную верстку
5. Добавить hover-эффекты для кнопок

## Технологии

- **HTML5** - структура макета
- **CSS3** - стилизация и анимации
- **Flexbox/Grid** - построение сетки кнопок
- **Адаптивная верстка** - поддержка мобильных устройств
