# Домашнее задание
## Веб-приложение "Обратная связь по курсу"

**Студент:** Ходина Александра Павловна  
**Группа:** ИУ5-41Б  

## Задание

1. Разработать веб-приложение для отображения групп помощи студентам
2. Реализовать все CRUD операции с использованием локального хранилища (store.js)
3. Добавить дополнительные функции: сравнение групп, проверка палиндрома
4. Интегрировать 3D модели для визуализации
5. Реализовать страницу с детальной информацией

## Технологии

- **HTML5** - структура приложения
- **CSS3** - стилизация и анимации
- **JavaScript (ES6+)** - бизнес-логика
- **Three.js** - 3D модели
- **Bootstrap 5** - адаптивный дизайн
